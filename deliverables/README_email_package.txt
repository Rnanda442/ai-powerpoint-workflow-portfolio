AI Geoscience Workflow Portfolio - Email Package

This package contains the Streamlit website files, curated data manifests, and visual assets.

How to run locally:

1. Unzip the package.
2. Open PowerShell in the unzipped folder.
3. Install dependencies if needed:
   pip install -r requirements.txt
4. Start the site:
   python -m streamlit run app.py --server.port 8502
5. Open:
   http://127.0.0.1:8502

Key pages:

- Overview
- Project Rooms
- Machine Learning Future
- Visual Gallery
- LinkedIn Evidence

Notes:

- The package includes copied visual assets used by the website.
- Some local source buttons point to files on the original Windows machine and may not open on another computer.
- The Processing earthquake LinkedIn post is verified, but the correct local 54-second video export still needs to be added before embedding it with sound.
